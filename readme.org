* chart : a configuration language for my preference

** Syntax
#+begin_src text
key   ::= symbol
value ::= <int> | <float> | <bool> | <string> | map | [value-1 value-2 ... value-n]
pair  ::= key = value
map   ::= { pair ... pair }
chart ::= pair ... pair
#+end_src
