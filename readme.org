* chart : a configuration language for my preference

** Syntax
#+begin_src text
chart ::= pair*
pair  ::= key = value
key   ::= symbol
value ::= <int> | <float> | <bool> | <string> | map | [ value* ]
map   ::= { chart }
#+end_src
